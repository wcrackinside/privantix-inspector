# Short Description (Hero)

---

# Product Overview

Privantix Catalog builds a portable and structured catalog from the technical analysis generated by Privantix Inspector.

The tool summarizes datasets, columns, metadata, rule detections and governance indicators into a human-readable and machine-readable inventory.

It allows organizations to quickly understand:

- what datasets exist
- where they are located
- which columns they contain
- whether potential sensitive data has been detected
- whether security metadata is available
- which rules were triggered during analysis

Privantix Catalog does not scan repositories directly.

It consumes the output generated by Privantix Inspector.

---

# Key Features

- Builds dataset catalog from inspector analysis
- Summarizes datasets and columns
- Flags datasets with potential sensitive data
- Includes checksum and security metadata indicators
- Generates JSON, CSV and HTML catalog outputs
- Fully portable CLI tool

# Typical Use Cases

## Data inventory

Organizations can build an inventory of datasets discovered in shared repositories.

## Governance visibility

Data governance teams can identify datasets containing potential sensitive information.

## Repository documentation

Teams can generate documentation for legacy data repositories.

## Audit preparation

Catalog reports help organizations prepare for data audits and compliance reviews.
